DONNÉES
📖 Guide — remplir les 3 flux
🔵 Polar → flux 1 (tout sur ce téléphone)
  1. App Polar Flow → autorise Health Connect (écriture).
  2. App FLINT → Profil → Synchroniser mon capteur.
  3. Après ~4 nuits : FLINT → Profil → Exporter (copie).
  4. Ici → Importer Polar → colle → Importer.
🟢 Whoop → flux 2 + 3 (via un ordi)
  1. Ordi : crée une app sur developer.whoop.com, remplis whoop-config.json.
  2. node whoop-sync.mjs login puis pull 30 → whoop-data.json.
  3. Envoie le fichier sur ton tél (Drive / mail).
  4. Ici → Importer whoop-data.json (remplit F2 et F3 d'un coup).
✋ Manuel (dépannage)
  1. Saisie ci-dessous → choisis la source (Polar/Whoop) + tape la nuit.
⚠️ Importe sur ce téléphone (données stockées sur l'appareil) · il faut ~4 nuits par source avant qu'un score s'affiche.
